>[!definition]
A list $v_1, v_2, ..., v_m$ of vectors is said to be dependent if there exists for $\lambda_1, \lambda_2, ..., \lambda_m \in \mathbf{F}$ a linear combination $\lambda_1v_1 + \lambda_2v_2 + ... + \lambda_mv_m = 0$ such that some $\lambda \neq 0$
